Module Dolmen_intf.Pretty

Pretty printing annotations

This module defines types to specify pretty printing annotations (such as associtativity, infix notations, etc...).

type name =
  1. | Exact of string
  2. | Renamed of string
type pos =
  1. | Infix
  2. | Prefix
type assoc =
  1. | Left
  2. | Right
type 'a print =
  1. | Ignore : _ print
  2. | P : (Stdlib.Format.formatter -> 'a -> unit) -> 'a print
